镜像¶

参考

模式: Object and 编辑模式s
菜单: Object/Mesh ‣ 镜像
快捷键: Ctrl-M

镜像ing an Object or Mesh selection will create a reversed version of the selection. The position of the mirrored version of the selection is determined by the 轴心点. A common use of mirroring is to model half an object, duplicate it and then use the mirror transform to create a reversed version to complete the model.

Note

镜像ed duplicates can also be created with a 镜像 Modifier.

镜像ing a selection.

用法¶

To mirror a selection along a particular global axis press: Ctrl-M, followed by X, Y or Z. The image 镜像ing a selection shows the results of this action after a mesh element has been duplicated.

In Mesh mode, you can mirror the selection on the currently selected 变换 Orientations by pressing the appropriate axis key a second time. For example, if the 变换 Orientation is set to Normal, pressing: Ctrl-M, followed by X and then X again will mirror the selection along the X-axis of the Normal Orientation.

Interactive mirror.

You can alternatively hold the MMB to interactively mirror the object by moving the mouse in the direction of the mirror axis.
