帮助系统¶

Blender has a range of built-in and web-based help options.

Tooltips¶

The Mouse pointer was Stopped for a while over the 渲染 Engines List in the 信息编辑器.

When hovering your cursor over a button or setting, it will show you the Tooltip, and related details depending on the button type, such as the key-shortcut for a tool.

The Python: text included last may be of interest to you if you’re an animator or automating tasks, see Scripting & Extending Blender.

Context Sensitive Manual Access¶

参考

模式: All modes
菜单: RMB, Online Manual
快捷键: Alt-F1

You may want to access help for a tool or area from within Blender.

Use the key-shortcut, or right-click menu to visit pages from this reference manual within Blender. This opens a webpage relating to the button under the cursor, supporting both tool and value buttons.

Note

We do not currently have 100% coverage, you may see an alert in the info header if some tools do not have a link to the manual.

其他 times buttons may link to more general sections of the documentation.

Help Menu¶

In the 信息编辑器 headers Help menu is a group of web links. The same links can also be found in the Slash Screen.

Tip

Browser and Internet Connection

Some forms of Help start up your web browser and access the Blender Foundation’s web servers. In order to do this, you must have configured a default web browser for your Operating System, and have a connection to the Internet.

Save System Info¶

Access Help ‣ Save System Info

This extracts system information which can be useful to include in bug reports, inspecting the configuration or diagnosing problems.

You will be prompted to save a text file system-info.txt.

The text file contains sections:

Blender
This section shows you the Blender version, details about the build configuration, and the path in which Blender is running.
Python
The Python version you are using, showing the paths of the Python programming language paths.
Directories

Paths used for scripts, data-files, presets and temporary files.

Those directories are configured using the 用户首选项 Editor.

OpenGL
This section shows the OpenGL version, the name of the manufacturer, and lists the capabilities of your hardware & driver.
