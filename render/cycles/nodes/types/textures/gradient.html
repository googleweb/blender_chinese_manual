Gradient 纹理¶

A gradient texture.

输入s¶

Vector
纹理 coordinate to sample texture at; defaults to Generated texture coordinates if the socket is left unconnected.

Properties¶

Type
The gradient can be Linear, Quadratic, Easing, Diagonal, Spherical, Quadratic Sphere or Radial.

Outputs¶

Color
纹理 color output.
Fac
纹理 intensity output.
