简介¶

This chapter explains how to change Blender’s default configuration with the 用户首选项 editor.

The Blender 用户首选项 editor contains settings to to control how Blender behaves.

打开用户首选项¶

To open the 用户首选项 editor and go to File ‣ 用户首选项.

配置¶

Now that you have opened the 用户首选项 editor, you can configure Blender to your liking. At the top of the editor, the available options are grouped into seven tabs:

界面
Change how UI elements are displayed and how they react.
Editing
Control how several tools will interact with your input.
Input
Customize how Blender reacts to the mouse and keyboard as well as define your own keymap.
Add-ons
Manage Blender’s Add-ons, allowing you to access features not built-in as well as install new features.
Themes
Customize interface appearance and colors.
File
配置 auto-save preferences and set default file paths for blend-files, rendered images, and more.
System
Set resolution, scripting console preferences, sound, graphics cards, and internationalization.

保存新的设置¶

Once you have set your preferences, you will need to manually save them, otherwise the new configuration will be lost after a restart. Blender saves its preferences to userpref.blend in your user folder (see next section, “恢复出厂设置”, for details).

In the 用户首选项 editor, click on the Save User 设置 button in the bottom left. This will save all of the new preferences.

恢复出厂设置¶

Go to File ‣ 恢复出厂设置 then save the preferences via the 用户首选项 editor.

Hint

It can be valuable to make a backup of your preferences the event that you lose your configuration.

See the directory layout section to see where your preferences are stored.

启动文件¶

参考

Mode: All modes
Menu: File ‣ Save 启动文件
Hotkey: Ctrl-U

When you start Blender or start a new project with the menu entry File ‣ New, a new scene is created from the default scene included with Blender.

This default scene can instead be your own customized setup.

To change the default scene, make all of the desired changes to the current scene or current file and File ‣ Save 启动文件.
