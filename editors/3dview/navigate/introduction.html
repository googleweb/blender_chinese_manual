简介¶

The 3D视图 is where you perform most of the object modeling and scene creation. Blender has a wide array of tools and options to support you in efficiently working with your mouse, keyboard and numpad.

Menus¶

The 3D视图 editor is comprised of a workspace and a header. The header is shown at the bottom or top of the workspace, and can be hidden if desired. The header shows you a menu and the current mode, as explained below.

3D视图 header.

View Menu¶

The View menu.

Properties Panel

Toggles the Properties side panel, which allows you to tweak many 3D视图 settings:

  • 变换
  • Grease Pencil
  • Display & View Panels
  • Background 图像s
  • 变换 Orientations
Tool Shelf
Toggles the Tool Shelf T, which appears on the left side of the 3D视图, and allows you to perform various operations, depending on the type of object selected, and the mode you are in.
相机
Switches the view to the current camera view.
Viewing angles:

These commands change the view to the default Top/Bottom, Front/Back, or Left/Right views.

  • Top Numpad7
  • Bottom Ctrl-Numpad7
  • Front Numpad1
  • Back Ctrl-Numpad1
  • Right Numpad3
  • Left Ctrl-Numpad3
相机s Menu:
Set Active object as camera Active camera
Perspective/Orthographic View
These commands change the projection of the 3D视图
Navigation Menu
This sub-menu contains commands for rotating and panning the view. Using these commands through the menu is not that efficient. However, like all Blender menus, the much more convenient keyboard shortcuts are listed next to the commands.
Align View

This submenu allows you to align the 3D视图 in certain ways.

  • Align to selected
  • Center cursor and view all
  • Align active camera to view
  • View Selected
  • Center View to cursor
Clipping Border
Allows you to define a clipping border to limit the 3D视图 display to a portion of 3D space.
Zoom Border
Allows you to define the area you want to zoom into.
Show all Layers
Makes all of the display layers visible.
Global View/Local View
Global view shows all of the 3D objects in the scene. Local view only displays the selected objects. This helps if there are many objects in the scene, that may be in the way. Accidentally pressing NumpadSlash can happen rather often if you are new to Blender, so if a bunch of the objects in your scene seem to have mysteriously vanished, try turning off local view.
View Selected
Zooms the 3D视图 to encompass all the selected objects.
View All
Zooms the 3D视图 to encompass all the objects in the current scene.
Play Back 动画
Plays back the animation from the current frame.
Duplicate area in new window
Clones the current 3D视图 in a new window.
Quad View
Toggles a four window 3D视图, each showing a different angle of the scene.
Toggle Full Screen
Maximizes the 3D视图 editor to fill the full screen area.

Lock to Scene

By default, the “lock” button to the right of the layer buttons is enabled. This means that in this view, the active layers and camera are those of the whole scene (and those used at render time). Hence, all 3D视图s locked this way will share the same active layers and camera - when you change them in one view, all locked others will immediately reflect these changes.

But if you disable this “lock” button, you then can specify different active layers and camera, specific to this view. This might be useful if you do not want to have your working areas (views) cluttered with the whole scene, and still have an ancillary complete view (which is unlocked with e.g. all layers shown). Or to have several views with different active cameras. Remember that you can use Ctrl-Numpad0 to make the active object the active camera.

Read more about Scenes
