物体模式s¶

Modes are a Blender-level object-oriented feature, which means that whole Blender application is always in a singular mode, and that the available modes vary depending on the selected active object’s type - most of them only enable the default 物体模式 (like cameras, lamps, etc.). Each mode is designed to edit an aspect of the selected object. See Tab. Blender’s Modes below for details.

The Mode drop-down list.

You set the current mode in the Mode drop-down list of 3D视图 header (see Fig. The Mode drop-down list.).

Warning

You can only select objects in 物体模式. In all others,the current object selection is “locked” (except, to some extent, with an armature’s 姿态模式).

Modes might affect many things in Blender:

  • They can modify the panels and/or controls available in some Properties editor tabs.
  • They can modify the behavior of the whole editor, like e.g. the UV/图像编辑器 and 3D视图.
  • They can modify the available header tools (menus and/or menu entries, as well as other controls...). For example, in the 3D视图 editor, the Object menu in 物体模式 changes to a Mesh menu in 编辑模式 (with an active mesh object!), and a Paint menu in 顶点绘制模式...
Blender’s Modes¶
Icon Name Shortcut Details
物体模式 None [1] The default mode, available for all object types, as it is dedicated to Object data-block editing (i.e. position/rotation/size).
编辑模式 Tab [1] A mode available for all renderable object types, as it is dedicated to their “shape” ObData data-block editing (i.e. vertices/edges/faces for meshes, control points for curves/surfaces, etc.)
雕刻模式 None [1] A mesh-only mode, that enables Blender’s mesh 3D-sculpting tool.
顶点绘制模式 None [1] A mesh-only mode, that allows you to set your mesh’s vertices colors (i.e. to “paint” them).
纹理绘制模式 None [1] A mesh-only mode, that allows you to paint your mesh’s texture directly on the model, in the 3D视图s.
权重绘制模式 Ctrl-Tab [2] A mesh-only mode, dedicated to vertex group weighting.
粒子模式 None [1] A mesh-only mode, dedicated to particle systems, useful with editable systems (hair).
姿态模式 Ctrl-Tab [2] An armature only mode, dedicated to armature posing.
编辑笔触模式 D-Tab. A Grease Pencil only mode, dedicated to editing Grease Pencil strokes.
[1](1, 2, 3, 4, 5, 6) Tab toggles 编辑模式.
[2](1, 2) Ctrl-Tab switches between the 权重绘制模式 (meshes)/姿态模式 (armatures) , and the other current one (by default, the 物体模式). However, the same shortcut has other, internal meanings in some modes (e.g. in 雕刻模式, it is used to select the current brush)...

As you can see, using shortcuts to switch between modes can become quite tricky, especially with meshes.

Note

The cursor becomes a brush in:

  • Vertex Paint mode
  • Weight Paint mode
  • 纹理 Paint mode.

We will not go into any more detail on modes usages here, However, most of them are tackled in the modeling chapter, as they are mainly related to this topic. The Particle mode is discussed in the particle section, and the 姿态模式 and 编辑模式 for armatures, in the rigging one.

Note

If you are reading this manual and some button or menu option is referenced that does not appear on your screen, it may be that you are not in the proper mode for that option to be valid.
