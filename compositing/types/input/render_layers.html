渲染 Layers Node¶

渲染 Layers Node.

This node is the starting place for getting a picture of your scene into the compositing node map.

输入s¶

This node has no input sockets.

Properties¶

Scene

Select the within your blend-file. The scene information taken is the raw footage (pre-compositing and pre-sequencing)

Hint

To use composited footage from another scene, it has to be rendered into a multilayer i.e. OpenEXR frameset as an intermediate file store and then imported with 图像 input node again.

渲染 layer
A list of available 渲染Layer. The render button is a short hand to re-render the active scene.

Outputs¶

图像
渲染ed image
Alpha
Alpha channel

渲染 passes sockets

Depending on the 渲染 passes that are enabled, other sockets are available. See Cycles render passes or Blender internal render passes.

Z
By default the Z depth pass is enabled.
