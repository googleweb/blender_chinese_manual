Map UV Node¶

Map UV Node.

With this node objects can be “re-textured” after they have been rendered.

To apply a texture to individual enumerated objects the ID Mask Node could be used.

Inputs¶

图像
The new 2D Texture.
UV
The input for UV render pass. See Cycles render passes or Blender internal render passes.

Hint

To store the UV pass a multilayer OpenEXR format could be used.

Properties¶

Alpha
Alpha control how much the new texture overlays the old.

Outputs¶

图像
The resulting image is the input image texture distorted to match the UV coordinates. That image can then be overlay mixed with the original image to paint the texture on top of the original.

Hint

When painting the new texture, it helps to have the UV maps for the original objects in the scene, it is recommended to keep those UV texture outlines around even, when shooting is done.

例子¶

In the example below, we have overlaid a grid pattern on top of the two “Emo” heads after they have been rendered.
