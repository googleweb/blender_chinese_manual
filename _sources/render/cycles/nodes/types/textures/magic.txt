
*************
Magic 纹理
*************

Psychedelic color texture.


输入s
======

Vector
   纹理 coordinate to sample texture at;
   defaults to Generated texture coordinates if the socket is left unconnected.
Scale
   Scale of the texture.
畸变
   Amount of distortion.


Properties
==========

Depth
   Number of iterations.


Outputs
=======

Color
   纹理 color output.
Fac
   纹理 intensity output.
