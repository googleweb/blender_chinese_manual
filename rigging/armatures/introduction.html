简介¶

An “armature” is a type of object used for rigging. Armature object borrows many ideas from real life skeletons.

Your first armature¶

In order to see what we’re talking about, let us try to add the default armature in Blender.

(Note that armature editing details are explained in the armatures editing section).

Open a default scene, then:

  • delete all objects in the scene
  • make sure the cursor is in the world origin with Shift-C
  • press Numpad1 to see the world in Front view
  • then, either: - in the Main Menu, go to Add ‣ Armature ‣ Single Bone - -or- in the 3D视图, add an armature with Shift-A pop-up ‣ Armature ‣ Single Bone
  • press NumpadDelete to see the armature at maximum zoom

The default armature Toolbox: –> Add Armature –> Single Bone

The armature object¶

As you can see, an armature is like any other object type in Blender:

  • It has a center, a position, a rotation and a scale factor.
  • It has an ObData data-block, that can be edited in 编辑模式.
  • It can be linked to other scenes, and the same armature data can be reused on multiple objects.
  • All animation you do in 物体模式 is only working on the whole object, not the armature’s bones (use the 姿态模式 to do this).

As armatures are designed to be posed, either for a static or animated scene, they have a specific state, called “rest position”. This is the armature’s default “shape”, the default position/rotation/scale of its bones, as set in 编辑模式.

In 编辑模式, you will always see your armature in rest position, whereas in 物体模式 and 姿态模式, you usually get the current “pose” of the armature (unless you enable the Rest Position button of the Armature panel).
